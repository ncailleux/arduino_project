## Description

### File *interact_led_panel.ino*

With that script you can create different action on a **MAX7219 Module**.
Interact with this module using a **Membrane Switch Module** & **Joystick Module**.

### Module connection
#### - MAX7219
	 VCC: 5V
	 GND: GND
	 DIN: 12 (digital)
	 CS: 10 (digital)
	 CLK: 11 (digital)

#### - Joystick
	 +5V: 5V
	 GND: GND
	 VRx: A0 (analog)
	 VRy: A1 (analog)
	 CLK: A2 (analog)

#### - Membrane Switch
	 2 (digital) to 9 (digital)
